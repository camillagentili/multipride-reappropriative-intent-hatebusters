# multipride-reappropriative-intent-hatebusters
Repository for the Hate Busters submission to the MultiPRIDE shared task on automatic identification of reappropriative intent in multilingual social media containing LGBTQ+ slurs. Includes preprocessing, BoW/TF-IDF and XLM-RoBERTa embeddings, linear and fine-tuned Transformer models, and code to reproduce the experiments.
